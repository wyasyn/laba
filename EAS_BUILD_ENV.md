# EAS Build environment variables

This project does not require a local `.env` file for development.

## Recommended workflow

Set variables on [expo.dev](https://expo.dev) under your project **Environment variables** for `development`, `preview`, and `production` when needed.

## Alternative workflow

Set values inside profile `env` blocks in [`eas.json`](../eas.json). Use this carefully and avoid committing sensitive values.
